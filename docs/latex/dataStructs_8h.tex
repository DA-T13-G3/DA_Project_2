{\footnotesize\ttfamily \mbox{\hyperlink{classEdge}{bool}} interfere (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{structweb}{web}}}]{a,  }\item[{\mbox{\hyperlink{structweb}{web}}}]{b }\end{DoxyParamCaption})}



Determines whether two webs interfere. 

Two webs interfere if their live ranges overlap at any program point. The function assumes that lines, start, and end vectors are sorted.

If the only shared line is one where\+:
\begin{DoxyItemize}
\item web A ends (due to a read), and
\item web B begins (due to a write), then the webs do not interfere.
\end{DoxyItemize}

\begin{DoxyReturn}{Returns}
True if the webs interfere, false otherwise. 
\end{DoxyReturn}
