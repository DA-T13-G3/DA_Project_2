\doxysubsection{Detailed Description}
Global parameters produced by the parser corresponding only to the registers input file. 

This structure contains all information extracted from the registers input file\+:


\begin{DoxyItemize}
\item alg\+: algorithm type and parameter.
\item regs\+: number of available registers.
\item valid\+: set to false if any parsing error occurs. 
\end{DoxyItemize}
