\doxysubsection{Detailed Description}
Global parameters produced by the parser. 

This structure contains all information extracted from the two input files\+:


\begin{DoxyItemize}
\item webs\+: list of all merged webs.
\item alg\+: algorithm type and parameter.
\item regs\+: number of available registers.
\item valid\+: set to false if any parsing error occurs. 
\end{DoxyItemize}
