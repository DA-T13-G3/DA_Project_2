Exports the finalized register allocation results to an output file. 

This function writes the contents of an allocation \textquotesingle{}result\textquotesingle{} structure into a clean, human-\/readable text format. It logs the consolidated metrics of all webs, reconstructs their exact lifetime intervals with specific boundary markers, and maps them to their allocated hardware registers or memory slots.

{\bfseries{Output format for the spilling and plitting algorithms\+:}} {\bfseries{Spilling\+:}} The webs spilled by the spilling algorithm will be commited into memory and this allocation will be represented by \textquotesingle{}M(spilled)\+: web\{id\}\textquotesingle{}. {\bfseries{Splitting\+:}} The webs that were split will have their id altered. Ex.\+: web0 becomes \doxylink{classEdge_ab976213419283ed754f42f57b195a288}{web0.\+0} and \doxylink{classEdge_ab976213419283ed754f42f57b195a288}{web0.\+1}.
