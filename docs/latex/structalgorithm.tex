\doxysubsection{Detailed Description}
Stores the algorithm configuration parsed from the second input file. 

The parser fills\+:
\begin{DoxyItemize}
\item type\+: which algorithm variant to use,
\item val\+: an integer parameter (max splits or max spills).
\end{DoxyItemize}

For "{}basic"{} and "{}free"{}, val = 0. For "{}splitting"{} and "{}spilling"{}, val is read from the input file.
