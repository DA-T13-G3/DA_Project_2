\doxysubsection{Detailed Description}
Final result of the register allocation algorithm. 

Contains\+:
\begin{DoxyItemize}
\item webs\+: the final list of webs (possibly modified by splitting).
\item assignments\+: groups of webs that share the same register.
\item regs\+: number of registers actually used.
\item possible\+: false if allocation failed (all webs must be spilled). 
\end{DoxyItemize}
