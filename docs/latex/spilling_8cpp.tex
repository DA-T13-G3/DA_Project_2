{\footnotesize\ttfamily \mbox{\hyperlink{classEdge}{struct}} \mbox{\hyperlink{structresult}{result}} spill (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classGraph}{Graph}}$<$ \mbox{\hyperlink{structweb}{web}} $>$ \texorpdfstring{$\ast$}{*}}]{g,  }\item[{\mbox{\hyperlink{structparams}{params}}}]{info,  }\item[{\mbox{\hyperlink{classEdge}{function}}$<$ \mbox{\hyperlink{classEdge}{int}}(\mbox{\hyperlink{classGraph}{Graph}}$<$ \mbox{\hyperlink{structweb}{web}} $>$ \texorpdfstring{$\ast$}{*}\mbox{\hyperlink{classEdge}{g}}, \mbox{\hyperlink{classEdge}{int}})$>$}]{coloring }\end{DoxyParamCaption})}



Builds a result struct while spilling a certain amount of webs into memory. 

This method extracts the graph\textquotesingle{}s vertices, isolates a strategic subset of size \textquotesingle{}X\textquotesingle{} containing the highest-\/degree candidates, and runs a binary search over the range of possible spill numbers. The search uses \doxylink{spilling_8cpp_ae74c014e1649f61b46847a6b0269a815}{attempt()} to find the smallest number of spills needed. Finally, it invokes \doxylink{spilling_8cpp_a0afd19877e83459263365b4c5c5a49be}{final\+Spill()} to commit the highest-\/efficiency combination and calls \doxylink{buildResult_8cpp_ae4a17a1faabc9d379ff778a1541406e8}{build()} to construct the final allocation report.

Details on the Heuristics Applied To make this function run in a practical timeframe, it tries to spill only a certain fixed number of combinations of webs, which significantly decreases its time complexity. It starts by trying to spill the nodes with the biggest degrees, as these are the ones most likely to increase the number of necessary registers, and then moves on to spilling nodes with lower degrees if that doesn\textquotesingle{}t work. This compromise trades unnecessary precision for a very significant increase in speed, which is why it was chosen.

{\bfseries{Time Complexity\+:}} O(Wlog(\+W) + I) where \textquotesingle{}W\textquotesingle{} is the number of webs and \textquotesingle{}I\textquotesingle{} is the total amount of interferences between webs
