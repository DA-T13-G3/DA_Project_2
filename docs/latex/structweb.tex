\doxysubsection{Detailed Description}
Represents a single live web of a variable. 

A web is the combination of one or more live ranges belonging to the same variable. The parser initially creates one web per live-\/range line in the input file. After parsing, \doxylink{parser_8cpp_ac1c18255324c0c59380f092569c4a9b7}{merge\+Webs()} merges all overlapping live ranges of the same variable into a single web.

Fields\+:
\begin{DoxyItemize}
\item var\+\_\+name\+: name of the variable.
\item start\+: program lines where the web begins (lines ending with \textquotesingle{}+\textquotesingle{}).
\item end\+: program lines where the web ends (lines ending with \textquotesingle{}-\/\textquotesingle{}).
\item lines\+: all program points where the variable is live.
\end{DoxyItemize}

All vectors are sorted after merging, and the \doxylink{dataStructs_8cpp_a3f5dffb50eb977e07687c21b4f4130d8}{interfere()} function assumes this ordering.
