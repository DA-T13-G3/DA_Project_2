Graph< web > create ( params  info)

Builds the graph from the set of webs.

This function constructs a graph where each web corresponds to a vertex. For every pair of webs, an interference edge is added if their live ranges overlap according to 'interfere()'. The resulting graph is the input for the register allocation algorithms.

Parameters
infoGlobal parameters containing the list of webs to insert in the graph.
Returns
A Graph<web> object representing the full interference graph.
