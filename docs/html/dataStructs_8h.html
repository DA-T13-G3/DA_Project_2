enum alg_type

Types of register allocation algorithms.

  • basic: Greedy graph coloring without modifications to the graph.
  • splitting: Iterative web splitting to reduce interference.
  • spilling: Web spilling (removal) when registers aren't enough.
  • free: Variant free for experimentation.

These values correspond directly to the "algorithm:" field in the second input file().

Determines whether two webs interfere.

Two webs interfere if their live ranges overlap at any program point. The function assumes that lines, start, and end vectors are sorted.

If the only shared line is one where:

  • web A ends (due to a read), and
  • web B begins (due to a write), then the webs do not interfere.
Returns
True if the webs interfere, false otherwise.
